## Features
1. Full roulette board with tap-to-enter keypad on every bet position
2. Animated spinning wheel with realistic ball sound
3. Simulation modes: Visual Spin, Rapid Sim, Timed Session
4. Exit conditions: Time limit, Loss limit, Win goal, Go Bust, Manual stop
5. Entry modes: Random, Manual entry
6. History tracking with session logs
7. Double confirmation on important actions
8. Payout reference card popup
9. Pre-loaded strategies (from catalog)
10. Bankroll management
11. Splash screen with Gamblers Anonymous (hold to reveal)
12. Simulation mode default, option for live mode
